# Stitch 3D Guess Who Duel 🎲✨

A modern, 3D isometric reimagining of the classic "Guess Who" board game. Built with Flutter, this project features a vibrant, interactive 3D interface, real-time multiplayer duels, and a sleek, polished UI.

## 🚀 Features
- **3D Isometric Board**: Immersive tabletop feel with standing cards that fold flat.
- **Real-time Multiplayer**: Duel with friends in synchronized matches.
- **Dynamic UI**: Vibrant animations, glassmorphism, and a premium design.
- **Custom Characters**: High-quality character data and imagery.
- **AI-Assisted Questions**: Integration for smarter gameplay.

## 🛠️ Built With
- **Flutter** - Cross-platform UI framework
- **Dart** - Programming language
- **Firebase/Custom Backend** - Real-time synchronization

## 📦 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/S0HAM03/stitch_3d_guess_who_duel.git
   ```
2. Navigate to the project folder:
   ```bash
   cd guess_who_3d
   ```
3. Get dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
